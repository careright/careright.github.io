Overview

This class represents a single record of custom import data from an external file. It is used to pre-process a CSV row before the import record is instantiated.

It can also be used to decide which import rows to select for the next stage of processing. Records are not selected for the next step unless selected is set to true.

When a triggerable class is fired from a Prerocess Record event, a variable named import_row of this class is predefined in the execution context.

All fields in the incoming CSV file are available via the getValue and setValue methods.
