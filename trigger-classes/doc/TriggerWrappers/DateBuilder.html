#build(value) ⇒ Date, null

Converts the parameter to a Date, if possible

The parameter can be a string in Australian (d/m/y) or International (Y-M-D) format, an existing Date, Time or DateTime. If passing a time in, the time part is removed.

Parameters:

  • value (Object)

Returns:

  • (Date)
  • (null) —

    if value could not be interpreted as a date

#from_js_date(date) ⇒ Date

Converts a native javascript Date to a Date object. The time part is ignored.

Returns:

  • (Date)

#today(_dummy) ⇒ Date

Returns a Date object for the current date. The timezone used will be the global setting Location Default Timezone. If that setting is blank, UTC will be used.

To return a date in the context of a specific location (which may use a different time zone), use Location.today()
