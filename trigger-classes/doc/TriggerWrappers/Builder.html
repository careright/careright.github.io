Overview

The builder classes provide helper methods for trigger scripts to create new objects.

This main Builder class has methods which return other builders as well as provider other helper methods.

#string_from_template(text, substitution) ⇒ String

Builds a string from a template

This method uses the Liquid templating language to construct a string based on either a trigger class or an object containing substitution values.

If a trigger class is provided, then its attributes are available to the template.

For more information on the Liquid templating language, refer to their <a target=“_blank” href=“websiteshopify.github.io/liquid/basics/introduction/”>website>.

Examples:

Parse a simple template

var result = builder.string_from_template('hi {{name}}', { name: 'Toby' }) // => 'Hi Toby'

Parameters:

  • text (String) —

    liquid template

  • substitution (Object) —

    an object or trigger class

Returns:

  • (String) —

    result
