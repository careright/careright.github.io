Instance Method Details

#search(query, empty_is_null = true) ⇒ LookupInstance, null

query is a hash or V8:Object with name:value lookup parameters Searches this lookup table for a value.

query is an object containing the search criteria, where the properties match key fields on the lookup table.

If multiple matches are found, any one could be returned. For this reason, make sure each key is used in the parameters.

Examples:

// Suppose a lookup table "Pricing" with keys fields "location" and
// "client_type" and value field "amount".
var lookup = builder.lookup.build('Pricing')
var result = lookup.search({ location: 'Clintel Clinic', client_type: 'MC' })
if (result) {
  console.log('pricing amount = ' + result.amount)
} else {
  console.log('No pricing amount for location Clintel Clinicl and client type MC')
}

Parameters:

  • query (Object)
  • empty_is_null (Boolean) (defaults to: true)

Returns:

    the record from the lookup tbale

    if no records were found
