#each(options, callback) ⇒ Object

Searches for assessments (note these are the assessment definitions, not the instances).

The function callback is called for each assessment which matches the criteria given in options.

This function only returns published and non-archived assessments. It will return disabled assessments as long as they are not archived.

Valid options: <ul> <li>name [String] - Only assessments with this name</li> <li>include_archived [Boolan] - if true, even archived assessments are returned</li> </ul>
