#ref_provider_name_invoice ⇒ String

Returns the name of the professional contact selected as the referral on this estimate. If there is no referral, the string Referral is returned.

The name is formatted as Mr Firstname LASTNAME.

Returns:

  • (String)

#ref_provider_no_invoice ⇒ String

Returns the provider number of the professional contact selected as the referral on this estimate. If there is no referral, the string not is returned.

#reference_invoice ⇒ String, null

Returns a reference invoice for this Estimate.

If the estimate has a service location, it returns a string made of the service location's Medicare Location ID plus the current time formatted as DDMMYYYYHHMM.

If the estimate does not have a service location, this property returns null.
