#build(task_type_code) ⇒ Task, null

Build a new (unsaved) task for a given task type. The code must be for a task type which is not disabled. The from_user will be set to the current user (if available).

Examples:

var task = builder.task.build('CODE')

  • task_type_code (String)

    if the task could not be built
