Overview

Represents a Case record.

Examples:

// Create a case at location 1
var my_case = patient.add_case('Alcohol', 1)
if (my_case) {
  // Open it
  if (!my_case.open()) {
    console.log('Failed to open case')
  }
  // Add an event for the medical provider 1
  var event = my_case.start_new_event(1)
  if (!event) {
    console.log('Failed to start a new event')
  }
} else {
  console.log('Failed to add case')
}

#start_new_event(staff_member) ⇒ Event, null

Creates and returns a new event associated with this case.

The parameter must be a MedicalProvider, Provider or a StaffMember object <strong>OR</strong> a number or a string containing the ID of a MedicalProvider.

The staff member provided as a parameter must have the role Can Edit Own Events.

The event will be created with Patient and Location copied from this case.

The returned event is already saved in the database.

Parameters:

  • staff_member (Object)
