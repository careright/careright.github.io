Overview

Represents an address belonging to a person/patient.

Addresses objects are usually accessed via a patient object using the home_address or postal_address methods.

If the patient doesn't have a home or postal address, then a new empty address object is created. To check if an address exists, look for a value in the address1 property.

Examples:

Retrieve first non-blank postal or home address

var address = patient.postal_address
if (!address.address1) {
  address = patient.home_address
}
if (!address.address1) {
  script.fatal('Patient has no postal or home address')
}

#quick_update(new_address1, new_address2, new_suburb, new_post_code, new_state_code, new_country_code) ⇒ Boolean

Updates and saves an address

State codes are strings like SA, NSW etc.

Country codes are from the Standard Australian Classification of Countries (SACC); Australia is 1100.

Examples:

Update a patient address

var addr = patient.home_address
if (addr.quick_update('1 Test St', null, 'SUBURB', '5000', 'SA', '1100')) {
  console.log('Address saved!')
} else {
  console.log('Address not saved')
}
