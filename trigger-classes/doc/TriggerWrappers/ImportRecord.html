Overview

This class represents a single record of custom import data from an external file, after any preprocessing has occurred.

When a triggerable class is fired from a Process Record event, a variable named import_record of this class is predefined in the execution context.

The variable will have properties matching the import elements defined on the Import definition.

Examples:

Accessing import record instance elements

// Assuming an import element named 'given_name'
var gn = import_record.given_name
// Replace the given name
import_record.given_name = 'Alex'

// Alternative syntax
var gn = import_record['given_name']
import_record['given_name'] = 'Alex'
