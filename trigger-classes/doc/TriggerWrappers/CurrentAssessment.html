Overview

Represents a CurrentAssessment record.

The CurrentAssessment object contains metadata related to an instance of an assessment. This includes the associated patient (for patient assessments), user who performed/approved and activity dates.

This object does not include the answers to questions from the assessment; for that you should access the associated AssessmentInstance class.
