#abn_invoice ⇒ String

If ABN is configured at Practice, this returns the Practice ABN from Accounting > Main choices. Otherwise it returns the ABN of the medical provider associated with this invoice.

#ref_provider_name_invoice ⇒ String

Returns the name of the professional contact selected as the referral on this invoice. If there is no referral, the string Referral is returned.

The name is formatted as Mr Firstname LASTNAME.

#reference_invoice ⇒ String, null

Returns a reference for this invoice.

If the invoice has a service location, it returns a string made of the service location's Medicare Location ID plus the current time formatted as DDMMYYYYHHMM.

If the invoice does not have a service location, this property returns null.

Returns:
