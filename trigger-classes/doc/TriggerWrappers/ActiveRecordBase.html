Overview

This is the parent class for all trigger classes which represent a CareRight record. It is not used directly, but provides methods which you can use on other classes.

#add_error(attr_or_msg = nil, message = nil) ⇒ Object

Adds a validation error to the given attribute with the optional message. If the attribute name is present, the error is associated with that attribute; otherwise it is associated with the record itself.

If an attribute is not provided, the message will be attached to the base of an object.

Examples:

Sets an error if a person's name is not Dave

if (patient.person.first_name.toLowerCase() == 'dave') {
  patient.person.add_error('first_name', 'not allowed to be Dave')
  patient.add_error('person', 'May not be called Dave')
}

Parameters:

  • attr_or_msg (String) (defaults to: nil) —

    field name OR message text

  • message (String) (defaults to: nil) —

    message text if a field name was provided
