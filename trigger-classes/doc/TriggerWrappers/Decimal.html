Overview

The Decimal class stores a number with a specific decimal precision.

Because Decimal works with a fixed precision, it does not suffer from rounding and approximation errors that can be introduced by using floating point numbers, including the native javascript.

If working with money, use the Money class instead, which is based on this class but fixed to 2 decimal places.
